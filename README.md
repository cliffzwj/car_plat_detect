# car_plat_detect
real time car plat detect use HyperLPR

使用开源HyperLPR，动态实时捕捉中国车牌号

其原理是利用cnn网络进行学习并逐帧处理视频流，可以使用摄像头捕捉车牌并同时识别车牌号码
